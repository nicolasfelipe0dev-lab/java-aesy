Exercício 1: Par ou Ímpar com Loops

Crie um programa que solicite ao usuário um número limite de iterações e, utilizando um laço de repetição for, imprima todos os números pares encontrados de 1 até o valor informado.

import java.util.Scanner; public class ParesIntervalo { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Digite um número limite positivo: "); int limite = scanner.nextInt(); System.out.println("Números pares encontrados:"); for (int i = 1; i <= limite; i++) { if (i % 2 == 0) { System.out.print(i + " "); } } System.out.println(); scanner.close(); } }

Exercício 2: Validador de Senha (while)

Construa um validador que impeça o fluxo do programa enquanto o usuário não digitar a senha textual correta ("JavaEasy123"). Quando acertar, imprima "Acesso Concedido".

import java.util.Scanner; public class LoginSistema { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); String senhaCorreta = "JavaEasy123"; String tentativa = ""; while (!tentativa.equals(senhaCorreta)) { System.out.print("Digite a senha de acesso: "); tentativa = scanner.next(); if (!tentativa.equals(senhaCorreta)) { System.out.println("Senha Incorreta! Tente novamente."); } } System.out.println("Acesso Concedido ao sistema!"); scanner.close(); } }

Exercício 3: Maior de Três Números

Escreva um programa que leia três números inteiros diferentes e determine qual deles é o maior utilizando condicionais encadeadas.

import java.util.Scanner; public class MaiorNumero { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Digite o primeiro número: "); int n1 = sc.nextInt(); System.out.print("Digite o segundo número: "); int n2 = sc.nextInt(); System.out.print("Digite o terceiro número: "); int n3 = sc.nextInt(); int maior = n1; if (n2 > maior) { maior = n2; } if (n3 > maior) { maior = n3; } System.out.println("O maior número digitado foi: " + maior); sc.close(); } }

Exercício 4: Calculadora Switch-Case

Desenvolva uma calculadora básica que solicite dois números reais e um caractere representando uma operação aritmética (`+`, `-`, `*`, `/`). Trate a operação usando switch.

import java.util.Scanner; public class CalculadoraSwitch { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Digite o primeiro número: "); double num1 = sc.nextDouble(); System.out.print("Digite o operador (+, -, *, /): "); char operador = sc.next().charAt(0); System.out.print("Digite o segundo número: "); double num2 = sc.nextDouble(); switch (operador) { case '+': System.out.println("Resultado: " + (num1 + num2)); break; case '-': System.out.println("Resultado: " + (num1 - num2)); break; case '*': System.out.println("Resultado: " + (num1 * num2)); break; case '/': if (num2 != 0) { System.out.println("Resultado: " + (num1 / num2)); } else { System.out.println("Erro: Divisão por zero!"); } break; default: System.out.println("Operador inválido!"); } sc.close(); } }

Exercício 6: Somatório com Flag de Parada

Escreva um programa que leia números inteiros continuamente do teclado. Quando o usuário digitar o número 0, o programa deve parar e exibir a soma total de todos os números digitados.

import java.util.Scanner; public class Somatorio { public static void main(String[] args) { Scanner sc = new Scanner(System.in); int soma = 0; int numero; do { System.out.print("Digite um número (0 para sair): "); numero = sc.nextInt(); soma += numero; } while (numero != 0); System.out.println("A soma total dos números digitados é: " + soma); sc.close(); } }

Exercício 7: Fatorial de um Número

Crie um algoritmo que receba um número inteiro positivo e calcule o seu fatorial utilizando uma estrutura de repetição decremental.

import java.util.Scanner; public class Fatorial { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Digite um número para calcular o fatorial: "); int num = sc.nextInt(); long fatorial = 1; for (int i = num; i >= 1; i--) { fatorial *= i; } System.out.println("O fatorial de " + num + " é " + fatorial); sc.close(); } }

Exercício 8: Verificador de Triângulos

Leia três valores decimais representando os lados de um triângulo. Primeiro, valide se eles formam um triângulo válido. Caso sim, classifique-o como: Equilátero, Isósceles ou Escaleno.

import java.util.Scanner; public class Triangulos { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Lado A: "); double a = sc.nextDouble(); System.out.print("Lado B: "); double b = sc.nextDouble(); System.out.print("Lado C: "); double c = sc.nextDouble(); if ((a < b + c) && (b < a + c) && (c < a + b)) { if (a == b && b == c) { System.out.println("Triângulo Equilátero (lados iguais)."); } else if (a == b || a == c || b == c) { System.out.println("Triângulo Isósceles (dois lados iguais)."); } else { System.out.println("Triângulo Escaleno (lados diferentes)."); } } else { System.out.println("Os valores fornecidos não formam um triângulo."); } sc.close(); } }

Exercício 9: Média de Notas com Validação

Construa um algoritmo que receba 4 notas escolares. O programa deve aceitar apenas valores entre 0.0 e 10.0. Caso seja inválido, peça novamente até que um número correto seja inserido.

import java.util.Scanner; public class ValidarNotas { public static void main(String[] args) { Scanner sc = new Scanner(System.in); double soma = 0; for (int i = 1; i <= 4; i++) { double nota; while (true) { System.out.print("Digite a nota " + i + " (0 a 10): "); nota = sc.nextDouble(); if (nota >= 0 && nota <= 10) { break; } System.out.println("Nota inválida! Digite novamente."); } soma += nota; } System.out.println("Média final: " + (soma / 4)); sc.close(); } }

Exercício 11: Média Populacional (Array Simples)

Crie um vetor (array) de inteiros com espaço para armazenar 5 idades. Leia as idades via console, guarde-as no array e use um laço para calcular a média de idade da população.

import java.util.Scanner; public class ArrayIdades { public static void main(String[] args) { Scanner sc = new Scanner(System.in); int[] idades = new int[5]; int soma = 0; for (int i = 0; i < idades.length; i++) { System.out.print("Digite a idade da pessoa " + (i + 1) + ": "); idades[i] = sc.nextInt(); soma += idades[i]; } double media = (double) soma / idades.length; System.out.println("A média de idade é: " + media + " anos."); sc.close(); } }

Exercício 14: Jogo da Adivinhação Básica

Tente novamente."); } else if (palpite > numeroSecreto) { System.out.println("Muito alto! Tente novamente."); } } System.out.println("Parabéns, você acertou!"); sc.close(); } }

Exercício 15: Cálculo de IMC com Classificação

Leia o peso e a altura de um indivíduo. Calcule o IMC ($peso / altura^2$) e apresente a classificação correspondente: Abaixo do peso (< 18.5), Normal (< 25) ou Sobrepeso (>= 25).

import java.util.Scanner; public class CalculoIMC { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Digite seu peso (kg): "); double peso = sc.nextDouble(); System.out.print("Digite sua altura (m): "); double altura = sc.nextDouble(); double imc = peso / (altura * altura); System.out.printf("Seu IMC é: %.2f\n", imc); if (imc < 18.5) { System.out.println("Classificação: Abaixo do peso."); } else if (imc < 25.0) { System.out.println("Classificação: Peso normal."); } else { System.out.println("Classificação: Sobrepeso."); } sc.close(); } }

Exercício 16: Inversor de Vetor

Preencha um array com 5 elementos inteiros inseridos pelo console. Imprima os elementos em ordem inversa utilizando decrementos em um loop.

import java.util.Scanner; public class VetorInverso { public static void main(String[] args) { Scanner sc = new Scanner(System.in); int[] numeros = new int[5]; for (int i = 0; i < 5; i++) { System.out.print("Digite o elemento " + i + ": "); numeros[i] = sc.nextInt(); } System.out.print("Ordem inversa: "); for (int i = numeros.length - 1; i >= 0; i--) { System.out.print(numeros[i] + " "); } System.out.println(); sc.close(); } }

Exercício 17: Múltiplos de 3 e 5

Escreva um programa que percorra os números de 1 a 30. Se for múltiplo de 3, imprima "Fizz"; se for múltiplo de 5, "Buzz"; se for de ambos, "FizzBuzz"; caso contrário, imprima o próprio número.

public class FizzBuzz { public static void main(String[] args) { for (int i = 1; i <= 30; i++) { if (i % 3 == 0 && i % 5 == 0) { System.out.println("FizzBuzz"); } else if (i % 3 == 0) { System.out.println("Fizz"); } else if (i % 5 == 0) { System.out.println("Buzz"); } else { System.out.println(i); } } } }

Exercício 18: Verificador de Maioridade em Grupo

Peça a idade de 5 pessoas. Ao final do processamento, exiba quantas pessoas já atingiram a maioridade penal (>= 18 anos) e quantas são menores de idade.

import java.util.Scanner; public class GrupoMaioridade { public static void main(String[] args) { Scanner sc = new Scanner(System.in); int maiores = 0, menores = 0; for (int i = 1; i <= 5; i++) { System.out.print("Idade da pessoa " + i + ": "); int idade = sc.nextInt(); if (idade >= 18) { maiores++; } else { menores++; } } System.out.println("Maiores de idade: " + maiores); System.out.println("Menores de idade: " + menores); sc.close(); } }

Exercício 19: Verificador de Número Primo

Construa um algoritmo que receba um número inteiro positivo e informe na tela se esse número é ou não um número primo (divisível apenas por 1 e por ele mesmo).

import java.util.Scanner; public class NumeroPrimo { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Digite um número inteiro: "); int num = sc.nextInt(); int divisores = 0; for (int i = 1; i <= num; i++) { if (num % i == 0) { divisores++; } } if (divisores == 2) { System.out.println("O número " + num + " é PRIMO."); } else { System.out.println("O número " + num + " NÃO é primo."); } sc.close(); } }

Exercício 20: Pesquisa de Menor Elemento

Leia 5 números inteiros via teclado e armazene-os em um array. Percorra o array utilizando estruturas condicionais dentro de um loop para descobrir e exibir o menor valor informado.

import java.util.Scanner; public class MenorValor { public static void main(String[] args) { Scanner sc = new Scanner(System.in); int[] valores = new int[5]; for (int i = 0; i < 5; i++) { System.out.print("Insira o valor " + (i + 1) + ": "); valores[i] = sc.nextInt(); } int menor = valores[0]; for (int i = 1; i < valores.length; i++) { if (valores[i] < menor) { menor = valores[i]; } } System.out.println("O menor valor inserido na lista foi: " + menor); sc.close(); } }
