Exercício 1: Média Aritmética

Escreva um programa em Java que receba três notas do tipo decimal (double) informadas pelo console, calcule e apresente a média final dessas notas.

import java.util.Scanner; public class MediaNotas { public static void main(String[] args) { Scanner teclado = new Scanner(System.in); System.out.print("Digite a primeira nota: "); double n1 = teclado.nextDouble(); System.out.print("Digite a segunda nota: "); double n2 = teclado.nextDouble(); System.out.print("Digite a terceira nota: "); double n3 = teclado.nextDouble(); double media = (n1 + n2 + n3) / 3.0; System.out.println("A média aritmética é: " + media); teclado.close(); } }

Exercício 2: Conversor de Idade

Crie um algoritmo que leia a idade de uma pessoa expressa em anos inteiros e calcule aproximadamente quantos dias de vida essa pessoa já viveu (considere anos fixos de 365 dias).

import java.util.Scanner; public class DiasVida { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Digite sua idade em anos: "); int idadeAnos = scanner.nextInt(); int totalDias = idadeAnos * 365; System.out.println("Você já viveu aproximadamente " + totalDias + " dias!"); scanner.close(); } }

Exercício 4: Área do Retângulo

import java.util.Scanner; public class AreaRetangulo { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Digite a base do retângulo: "); double base = sc.nextDouble(); System.out.print("Digite a altura do retângulo: "); double altura = sc.nextDouble(); double area = base * altura; System.out.println("A área do retângulo é: " + area); sc.close(); } }

Exercício 5: Conversor de Temperatura

Leia uma temperatura em graus Celsius (°C) e converta-a para Fahrenheit (°F). (Fórmula: $F = C \times 1.8 + 32$).

import java.util.Scanner; public class ConversorTemperatura { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Digite a temperatura em °C: "); double c = sc.nextDouble(); double f = c * 1.8 + 32; System.out.println("A temperatura em Fahrenheit é: " + f + "°F"); sc.close(); } }

Exercício 9: Calculadora de Descontos

Faça um algoritmo que leia o preço de um produto e mostre seu novo preço com 5% de desconto.

import java.util.Scanner; public class CalculadoraDesconto { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Preço original do produto: R$ "); double preco = sc.nextDouble(); double desconto = preco * 0.05; double precoFinal = preco - desconto; System.out.println("Valor do desconto: R$ " + desconto); System.out.println("Preço final a pagar: R$ " + precoFinal); sc.close(); } }

Exercício 11: Conversor de Dólar

import java.util.Scanner; public class ConversorMoeda { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Quantos reais você tem na carteira? R$ "); double reais = sc.nextDouble(); double dolares = reais / 5.00; System.out.printf("Você pode comprar: US$ %.2f\n", dolares); sc.close(); } }

Exercício 12: Média Ponderada

Calcule a média ponderada de duas notas. A primeira nota tem peso 2 e a segunda nota tem peso 3. Peça as duas notas ao usuário.

import java.util.Scanner; public class MediaPonderada { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Digite a nota 1: "); double nota1 = sc.nextDouble(); System.out.print("Digite a nota 2: "); double nota2 = sc.nextDouble(); double mediaPonderada = ((nota1 * 2) + (nota2 * 3)) / (2 + 3); System.out.println("A média ponderada é: " + mediaPonderada); sc.close(); } }

Exercício 13: Cálculo de Aluguel de Carros

Um carro alugado custa R$ 60 por dia e R$ 0,15 por km rodado. Escreva um programa que pergunte a quantidade de km percorridos e a quantidade de dias pelos quais ele foi alugado. Calcule o preço total a pagar.

import java.util.Scanner; public class AluguelCarro { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Dias de aluguel: "); int dias = sc.nextInt(); System.out.print("Km rodados: "); double km = sc.nextDouble(); double total = (dias * 60) + (km * 0.15); System.out.println("Total a pagar: R$ " + total); sc.close(); } }

Exercício 19: Volume de Caixa

Peça os três dados necessários para calcular o volume de uma caixa retangular: comprimento, largura e altura. Aplique a fórmula matemática padrão ($volume = comprimento \times largura \times altura$).

import java.util.Scanner; public class VolumeCaixa { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Digite o comprimento da caixa: "); double comprimento = sc.nextDouble(); System.out.print("Digite a largura da caixa: "); double largura = sc.nextDouble(); System.out.print("Digite a altura da caixa: "); double altura = sc.nextDouble(); double volume = comprimento * largura * altura; System.out.println("O volume total da caixa é de: " + volume); sc.close(); } }

Exercício 20: Desconto de Previdência

Receba o salário bruto de um funcionário. Calcule e apresente o valor final líquido após deduzir uma taxa previdenciária padrão e fixa de 11%.

import java.util.Scanner; public class DescontoPrevidencia { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Digite o salário bruto do colaborador: R$ "); double bruto = sc.nextDouble(); double desconto = bruto * 0.11; double liquido = bruto - desconto; System.out.println("Valor do desconto (11% INSS): R$ " + desconto); System.out.println("Salário líquido final: R$ " + liquido); sc.close(); } }
