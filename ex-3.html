Exercício 1: Modelagem de Conta Bancária Simples

Crie uma classe independente chamada ContaBancaria com encapsulamento adequado (atributos private: titular, saldo). Desenvolva métodos para depositar, sacar (validando fundos) e exibir dados estruturados.

// Arquivo: ContaBancaria.java public class ContaBancaria { private String titular; private double saldo; public ContaBancaria(String titular, double saldoInicial) { this.titular = titular; this.saldo = saldoInicial; } public void depositar(double valor) { if (valor > 0) { this.saldo += valor; System.out.println("Depósito de R$" + valor + " realizado."); } } public void sacar(double valor) { if (valor > 0 && valor <= this.saldo) { this.saldo -= valor; System.out.println("Saque de R$" + valor + " concluído."); } else { System.out.println("Falha no saque: Saldo insuficiente."); } } public void exibirStatus() { System.out.println("Titular: " + titular + " | Saldo Atual: R$" + saldo); } }

Exercício 2: Herança e Polimorfismo Básicos

Crie uma classe abstrata base Funcionario com um método abstrato calcularSalario(). Implemente duas subclasses: Gerente (salário fixo + bônus) e Desenvolvedor (remuneração por hora trabalhada).

// Classe Base Abstrata abstract class Funcionario { protected String nome; public Funcionario(String nome) { this.nome = nome; } public abstract double calcularSalario(); } // Subclasse Gerente class Gerente extends Funcionario { private double salarioBase; private double bonus; public Gerente(String nome, double salarioBase, double bonus) { super(nome); this.salarioBase = salarioBase; this.bonus = bonus; } @Override public double calcularSalario() { return salarioBase + bonus; } } // Subclasse Desenvolvedor class Desenvolvedor extends Funcionario { private double valorHora; private int horasTrabalhadas; public Desenvolvedor(String nome, double valorHora, int horasTrabalhadas) { super(nome); this.valorHora = valorHora; this.horasTrabalhadas = horasTrabalhadas; } @Override public double calcularSalario() { return valorHora * horasTrabalhadas; } }

Exercício 4: Sobrecarga de Métodos (Overloading)

Crie uma classe útil chamada CalculadoraArea. Implemente métodos estáticos homônimos chamados calcularArea sobrecarregados para calcular a área de um Quadrado (1 parâmetro real), Retângulo (2 parâmetros reais) e Círculo (1 parâmetro real modificando a assinatura interna).

public class CalculadoraArea { // Área do Quadrado public static double calcularArea(double lado) { return lado * lado; } // Área do Retângulo public static double calcularArea(double base, double altura) { return base * altura; } // Área do Círculo (usando parâmetro int para diferenciar do quadrado) public static double calcularArea(double raio, boolean ehCirculo) { if (ehCirculo) { return Math.PI * Math.pow(raio, 2); } return 0; } }

Exercício 5: Manipulação Dinâmica com ArrayList de Objetos

Construa um mini-sistema de Gerenciamento de Alunos. Crie uma classe Aluno (nome, nota) e gerencie múltiplos alunos inseridos via laço dinâmico usando um ArrayList<Aluno>, exibindo a lista e a média geral da turma ao final.

import java.util.ArrayList; import java.util.Scanner; class Aluno { String nome; double nota; public Aluno(String nome, double nota) { this.nome = nome; this.nota = nota; } } public class CadastroTurma { public static void main(String[] args) { Scanner sc = new Scanner(System.in); ArrayList<Aluno> lista = new ArrayList<>(); while (true) { System.out.print("Nome do aluno (ou 'sair'): "); String nome = sc.next(); if (nome.equalsIgnoreCase("sair")) break; System.out.print("Nota: "); double nota = sc.nextDouble(); lista.add(new Aluno(nome, nota)); } double soma = 0; for (Aluno a : lista) { System.out.println("Aluno: " + a.nome + " - Nota: " + a.nota); soma += a.nota; } System.out.println("Média Geral: " + (lista.isEmpty() ? 0 : (soma / lista.size()))); sc.close(); } }

Exercício 6: Tratamento Avançado de Exceções (try-catch)

Crie um programa que execute a divisão de dois inteiros colhidos por console. Capture de forma isolada e elegante as exceções aritméticas (como divisão por zero) e de tipos incorretos de dados inseridos.

import java.util.InputMismatchException; import java.util.Scanner; public class TratamentoErros { public static void main(String[] args) { Scanner sc = new Scanner(System.in); try { System.out.print("Numerador: "); int num = sc.nextInt(); System.out.print("Denominador: "); int den = sc.nextInt(); int resultado = num / den; System.out.println("Resultado: " + resultado); } catch (ArithmeticException e) { System.out.println("Erro Crítico: Impossível dividir por zero!"); } catch (InputMismatchException e) { System.out.println("Erro de Entrada: Você precisa digitar apenas números inteiros."); } finally { sc.close(); } } }

public class Usuario { private String nome; private String email; public Usuario(String nome, String email) { this.nome = nome; this.email = email; } @Override public boolean equals(Object obj) { if (this == obj) return true; if (obj == null || getClass() != obj.getClass()) return false; Usuario outro = (Usuario) obj; return this.email.equalsIgnoreCase(outro.email); } }

Exercício 20: Estrutura de Coleção Chave-Valor (HashMap)

import java.util.HashMap; import java.util.Scanner; public class CatalogoProdutos { public static void main(String[] args) { HashMap<String, Double> produtos = new HashMap<>(); produtos.put("Notebook", 4500.0); produtos.put("Smartphone", 2200.0); produtos.put("Teclado", 250.0); Scanner sc = new Scanner(System.in); System.out.print("Digite o nome do produto para consulta: "); String busca = sc.next(); if (produtos.containsKey(busca)) { System.out.println("Preço de " + busca + ": R$ " + produtos.get(busca)); } else { System.out.println("Produto não localizado no estoque."); } sc.close(); } }
