Recursos Nativos do Java (API Padrão)

O Java já vem de fábrica com um ecossistema gigante. Conheça os principais pacotes e classes essenciais da linguagem que você pode usar imediatamente sem instalar nada.

Classe Math

Fornece constantes e métodos estáticos para operações matemáticas fundamentais como potências, raízes, trigonometria e arredondamentos.

// Nativo (java.lang.Math)

Classe String

Provê manipulação avançada de textos, busca de caracteres, divisão de frases, conversões de caixa e comparações textuais seguras.

// Nativo (java.lang.String)

Scanner

Essencial para capturar e converter dados digitados pelo usuário no console ou ler arquivos de texto básicos.

import java.util.Scanner;

Collections (ArrayList)

Gerencia estruturas de dados dinâmicas e inteligentes, permitindo criar listas cujo tamanho cresce e diminui de forma flexível.

import java.util.ArrayList;

Java IO / NIO

Conjunto de ferramentas robustas para criar, ler, gravar e mover arquivos de texto ou binários dentro do seu sistema de arquivos.

import java.io.File;

Java Time API

API moderna para controle preciso de fusos horários, cálculo de períodos, formatação de datas locais e cronômetros do sistema.

import java.time.LocalDate;

Random

Gerador pseudoaleatório nativo capaz de sortear números inteiros, decimais ou valores booleanos para dinâmicas de jogos ou testes.

import java.util.Random;

Java Swing

Conjunto de componentes gráficos legado e nativo para criar janelas desktop, botões, caixas de diálogo e interfaces completas.

import javax.swing.JFrame;
