Desafios com APIs Nativas

Aprenda a aplicar métodos avançados nativos das classes utilitárias do ecossistema padrão da JDK.

1. Classe java.lang.Math

Exercício 1: Cálculos de Geometria Estática (Math)

Construa um programa que receba o raio de uma circunferência, calcule e exiba sua área total. Use a constante de Pi e a função de potência contidas na classe nativa java.lang.Math.

import java.util.Scanner; public class AreaCirculo { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Digite o raio do círculo: "); double raio = scanner.nextDouble(); // Uso direto da API nativa Math double area = Math.PI * Math.pow(raio, 2); System.out.printf("A área total do círculo é: %.2f\n", area); scanner.close(); } }

Exercício 2: Arredondamentos Mecânicos

Leia um número decimal via console e exiba três resultados usando Math: o valor arredondado para cima (teto), o valor arredondado para baixo (piso) e a raiz quadrada absoluta do número.

import java.util.Scanner; public class Arredondamentos { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Digite um número decimal: "); double num = scanner.nextDouble(); System.out.println("Arredondado para cima (ceil): " + Math.ceil(num)); System.out.println("Arredondado para baixo (floor): " + Math.floor(num)); System.out.println("Raiz quadrada (sqrt): " + Math.sqrt(Math.abs(num))); scanner.close(); } }

Exercício 3: O Maior e o Menor Valor

Solicite dois números inteiros ao usuário. Sem utilizar nenhuma estrutura condicional (if ou operador ternário), descubra e exiba qual é o maior e qual é o menor número usando os métodos nativos da Math.

import java.util.Scanner; public class MaiorMenorMath { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Digite o primeiro inteiro: "); int n1 = scanner.nextInt(); System.out.print("Digite o segundo inteiro: "); int n2 = scanner.nextInt(); int maior = Math.max(n1, n2); int menor = Math.min(n1, n2); System.out.println("Maior número: " + maior); System.out.println("Menor número: " + menor); scanner.close(); } }

2. Classe java.lang.String

Exercício 4: Verificador de Sobrenome

Receba o nome completo de uma pessoa via teclado. Converta a String para letras maiúsculas e utilize um método nativo para verificar se o nome termina com o sobrenome "SILVA".

import java.util.Scanner; public class VerificadorSilva { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Digite seu nome completo: "); String nomeCompleto = scanner.nextLine(); String nomeMaiusculo = nomeCompleto.toUpperCase(); boolean ehSilva = nomeMaiusculo.endsWith("SILVA"); System.out.println("O nome termina com 'SILVA'? " + ehSilva); scanner.close(); } }

Exercício 5: Substituição de Caracteres

Escreva um programa que receba uma frase qualquer e altere todas as ocorrências de espaços em branco pelo caractere underscore (_) utilizando o método de substituição nativo da classe String.

import java.util.Scanner; public class SubstituiEspacos { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Digite uma frase: "); String frase = scanner.nextLine(); String fraseModificada = frase.replace(" ", "_"); System.out.println("Frase modificada: " + fraseModificada); scanner.close(); } }

Exercício 6: Divisão de Strings (Tokenização)

Solicite ao usuário que insira uma lista de compras de supermercado com itens separados por vírgula (ex: "Arroz,Feijão,Macarrão"). Utilize o método split() para quebrar a String e exiba cada item em uma linha diferente.

import java.util.Scanner; public class DivisorItens { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Digite os itens separados por vírgula: "); String entrada = scanner.nextLine(); String[] itens = entrada.split(","); System.out.println("Seus itens separados:"); for (String item : itens) { System.out.println("- " + item.trim()); } scanner.close(); } }

3. API de Datas (java.time)

Exercício 7: Idade Precisa com LocalDate

Leia o ano, o mês e o dia de nascimento de uma pessoa. Converta para LocalDate e calcule a quantidade exata de anos de idade que ela possui hoje usando a classe utilitária ChronoUnit ou Period.

import java.time.LocalDate; import java.time.Period; import java.util.Scanner; public class CalculadorIdade { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Ano de nascimento (AAAA): "); int ano = scanner.nextInt(); System.out.print("Mês de nascimento (MM): "); int mes = scanner.nextInt(); System.out.print("Dia de nascimento (DD): "); int dia = scanner.nextInt(); LocalDate dataNascimento = LocalDate.of(ano, mes, dia); LocalDate dataAtual = LocalDate.now(); int idade = Period.between(dataNascimento, dataAtual).getYears(); System.out.println("Sua idade exata é: " + idade + " anos."); scanner.close(); } }

Exercício 8: Formatador de Data e Hora (DateTimeFormatter)

Capture o momento exato do sistema utilizando LocalDateTime. Imprima o resultado original e, em seguida, aplique a formatação nativa para exibi-lo no padrão brasileiro: "dd/MM/yyyy HH:mm:ss".

import java.time.LocalDateTime; import java.time.format.DateTimeFormatter; public class FormatadorData { public static void main(String[] args) { LocalDateTime agora = LocalDateTime.now(); System.out.println("Formato ISO Padrão: " + agora); DateTimeFormatter formatador = DateTimeFormatter.ofPattern("dd/MM/yyyy HH:mm:ss"); String dataFormatada = agora.format(formatador); System.out.println("Formato Nacional: " + dataFormatada); } }

Exercício 9: Vencimento de Parcelas

Imagine um sistema financeiro. Registre a data de uma compra (Data Atual). Adicione artificialmente 3 meses e 15 dias a essa data usando métodos de manipulação temporal e exiba o dia exato do vencimento da fatura.

import java.time.LocalDate; public class VencimentoFatura { public static void main(String[] args) { LocalDate dataCompra = LocalDate.now(); System.out.println("Data da Compra: " + dataCompra); // Adicionando tempo à data nativamente LocalDate vencimento = dataCompra.plusMonths(3).plusDays(15); System.out.println("Data Limite de Vencimento: " + vencimento); } }

4. Classe java.util.ArrayList

Exercício 10: CRUD Básico de Nomes

Instancie um ArrayList<String>. Adicione 4 nomes ao carrinho. Remova o segundo elemento da lista pelo seu índice físico e verifique se o nome "Ana" consta no vetor restante utilizando o método nativo de checagem.

import java.util.ArrayList; public class ListaNomes { public static void main(String[] args) { ArrayList<String> nomes = new ArrayList<>(); nomes.add("Carlos"); nomes.add("Bruno"); nomes.add("Ana"); nomes.add("Daniel"); nomes.remove(1); // Remove "Bruno" boolean temAna = nomes.contains("Ana"); System.out.println("A lista possui a 'Ana'? " + temAna); System.out.println("Tamanho atual da lista: " + nomes.size()); } }

Exercício 11: Ordenação de Listas Dinâmicas (Collections)

Leia do teclado 5 números inteiros desordenados e armazene-os em um ArrayList. Use o método estático nativo da classe java.util.Collections para ordenar a lista em ordem crescente e exiba o resultado.

import java.util.ArrayList; import java.util.Collections; import java.util.Scanner; public class OrdenaLista { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); ArrayList<Integer> numeros = new ArrayList<>(); for (int i = 0; i < 5; i++) { System.out.print("Insira o número " + (i + 1) + ": "); numeros.add(scanner.nextInt()); } Collections.sort(numeros); // Ordenação nativa ascendente System.out.println("Lista Ordenada: " + numeros); scanner.close(); } }

Exercício 12: Substituição Condicional em Listas

Crie uma lista com 5 números inteiros mistos. Percorra a lista e, para cada número negativo encontrado, substitua o valor existente por zero usando o método set() da estrutura do ArrayList.

import java.util.ArrayList; public class LimpaNegativos { public static void main(String[] args) { ArrayList<Integer> valores = new ArrayList<>(); valores.add(10); valores.add(-5); valores.add(23); valores.add(-12); valores.add(7); for (int i = 0; i < valores.size(); i++) { if (valores.get(i) < 0) { valores.set(i, 0); // Substituição indexada } } System.out.println("Lista limpa sem valores negativos: " + valores); } }

5. Classe java.util.HashMap

Exercício 13: Dicionário de Sinônimos

Crie um mapa chave-valor associando palavras aos seus sinônimos (ex: "Feliz" -> "Alegre"). Solicite que o usuário digite uma palavra e apresente o sinônimo direto cadastrado no mapa.

import java.util.HashMap; import java.util.Scanner; public class SinonimosMap { public static void main(String[] args) { HashMap<String, String> dicionario = new HashMap<>(); dicionario.put("rapido", "veloz"); dicionario.put("feliz", "alegre"); dicionario.put("belo", "bonito"); Scanner scanner = new Scanner(System.in); System.out.print("Digite uma palavra para buscar o sinônimo: "); String busca = scanner.next().toLowerCase(); if (dicionario.containsKey(busca)) { System.out.println("O sinônimo de " + busca + " é " + dicionario.get(busca)); } else { System.out.println("Palavra não encontrada no dicionário."); } scanner.close(); } }

Exercício 14: Controle de Estoque com Remoção

Aloque um HashMap mapeando o nome de produtos (Chave) à quantidade de unidades disponíveis no estoque (Valor). Remova um item completamente do mapa caso a quantidade em estoque atinja o valor zero.

import java.util.HashMap; public class GerenciadorEstoque { public static void main(String[] args) { HashMap<String, Integer> estoque = new HashMap<>(); estoque.put("Monitor", 12); estoque.put("Mouse", 0); // Sem estoque estoque.put("Teclado", 5); // Removendo do mapa se o valor for zero estoque.remove("Mouse"); System.out.println("Catálogo ativo de estoque: " + estoque); } }

Exercício 15: Listagem Completa de Chaves

Popule um mapa associando o CPF de 3 cidadãos aos seus respectivos nomes completos. Utilizando o método nativo keySet(), extraia e imprima apenas a lista contendo os CPFs cadastrados no sistema.

import java.util.HashMap; import java.util.Set; public class ListadorChaves { public static void main(String[] args) { HashMap<String, String> usuarios = new HashMap<>(); usuarios.put("111.222.333-44", "Rodrigo Alves"); usuarios.put("555.666.777-88", "Fernanda Costa"); // Coleta o conjunto exclusivo de chaves Set<String> cpfs = usuarios.keySet(); System.out.println("Lista de CPFs registrados na base de dados:"); for (String cpf : cpfs) { System.out.println("-> " + cpf); } } }

Exercício 17: Moeda Virtual (Cara ou Coroa)

Crie uma simulação de "Cara ou Coroa". Use a classe Random com o método de retorno booleano (nextBoolean()). Se o sorteio retornar verdadeiro, imprima "Cara", senão, exiba "Coroa".

import java.util.Random; public class CaraCoroa { public static void main(String[] args) { Random rand = new Random(); boolean sorteio = rand.nextBoolean(); if (sorteio) { System.out.println("Resultado: CARA!"); } else { System.out.println("Resultado: COROA!"); } } }

Exercício 18: Sorteador de Índices de Arrays

Defina um vetor fixo contendo 4 prêmios (ex: "Notebook", "Vale Compras", "Bicicleta", "Ingresso"). Utilize a classe Random para sortear um índice numérico válido e exiba o prêmio ganho pelo usuário.

import java.util.Random; public class SorteioPremios { public static void main(String[] args) { String[] premios = {"Notebook", "Vale Compras de R$500", "Bicicleta Ergométrica", "Ingresso de Cinema"}; Random rand = new Random(); // Limita o sorteio dinamicamente ao tamanho máximo do vetor int indiceSorteado = rand.nextInt(premios.length); System.out.println("Parabéns! Você foi sorteado e ganhou: " + premios[indiceSorteado]); } }
